# Internship Management System

## Overview
A simple Internship Management System built for learning Frappe Framework and ERPNext v15 customization.

## Features
- Department Management
- Mentor Management
- Intern Management
- Task Assignment
- Workspace Dashboard
- Reports

- Description

### Intern

- Email

- Status

### Task

- Intern

- Status
